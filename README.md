# whiteshoeswednesday
keep track of your whiteshoeswednesday streak

[whiteshoeswednesday.com](whiteshoeswednesday.com)

## run
run angular app
```bash
ng serve
```

run node app
```bash
node app.js
```

### requirements
 configs.js
```javascript
module.exports = {
  url: "mongodb://localhost:27017",
  recaptcha_secret_key: "YOUR_RECAPTCHA_SECRET_KEY"
}
```
